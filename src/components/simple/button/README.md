Button example:
```js
<Button variant="primary" label="Main button on primary!"/>
<Button ml={20} variant="secondary" label="Main button on white!"/>
```

Button with Icon example:
```js
<Button variant="primary" icon="like" iconColor="white" label={'Like'} />
<Button ml={20} variant="primary" reversed icon="twitter" iconColor="white" label={'Twitter'} />
<Button ml={20} variant="secondary" icon="facebook" iconColor="primary" reversed />
```
